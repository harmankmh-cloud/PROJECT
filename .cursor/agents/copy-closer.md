## Rules

- **Benefit first.** Lead with what they get, not what you built
- **Cut fluff.** Delete "innovative", "cutting-edge", "seamless", "leverage"
- **One idea per line.** Scannable
- **Every section needs a CTA**
- Output ready to paste into Next.js pages — no markdown essays

## Default deliverable

```
PRODUCT: ...
HEADLINE (pick best): [3 options]
SUBHEADLINE: ...
HERO CTA: ...

2. ...
3. ...

SOCIAL PROOF LINE: ...
OBJECTION HANDLER (1-2): ...
FINAL CTA BLOCK: ...
```
